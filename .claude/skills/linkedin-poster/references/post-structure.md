# LinkedIn Post Structure & Templates

## Post Anatomy

```
[Hook — 1-2 lines, bold claim, question, or surprising fact]

[Context — what happened / what you built, 2-3 lines]

[Value — what the reader learns or gains, 2-3 lines]

[CTA — question to drive comments, 1 line]

#Tag1 #Tag2 #Tag3 #Tag4
```

## Hook Patterns by Post Type

### Achievement/Milestone
```
We just hit [result] in [timeframe]. Here's what actually made the difference.
```

### Lesson Learned
```
I made a mistake that cost me [outcome]. Here's what I learned.
```

### Build in Public
```
What we built this week with AI automation (and why it matters for small teams).
```

### Insight/Tip
```
Most founders waste [X hours] on [task]. Here's how to get it back.
```

## Character Count Guide

| Section | Target |
|---|---|
| Hook | 80-120 chars |
| Context + Value | 400-600 chars |
| CTA | 60-100 chars |
| Hashtags | 60-100 chars |
| **Total** | **< 1,300 chars** |
