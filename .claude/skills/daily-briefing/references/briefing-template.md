# CEO Briefing Template

Use this exact structure when writing to `/Vault/Briefings/{YYYY-MM-DD}_Briefing.md`.

```markdown
---
generated: {iso_timestamp}
period: {start_date} to {end_date}
---

# Monday Morning CEO Briefing

## Executive Summary
{1-2 sentences: was it a good week? key highlight or concern}

## Activity This Week
- **Tasks completed:** {count}
- **Pending actions:** {count}
- **Emails processed:** {count}
- **Plans overdue (>48h):** {count}

## Completed Tasks
{list each Done file from this week — one line each with date}

## Bottlenecks
{table of any Plans older than 48h still pending}

| Plan File | Created | Hours Overdue |
|---|---|---|
| {filename} | {date} | {hours} |

(Write "None identified" if table is empty)

## Revenue & Goals
{read from Business_Goals.md and summarize MTD vs target}

- **Monthly goal:** {value}
- **Current MTD:** {value} ({percent}% of target)
- **Trend:** On track / Behind / Ahead

## Proactive Suggestions
{based on patterns — overdue invoices, unused subscriptions, upcoming deadlines}

- **{item}:** {observation}. [ACTION] {suggested action}?

## Upcoming Deadlines (next 14 days)
{read from Business_Goals.md Active Projects section}

---
*Generated by AI Employee — review daily*
```

## Dashboard.md Update

**Last Briefing:** {YYYY-MM-DD} — {one line executive summary}
