# Company Handbook - Rules of Engagement

## Communication Rules

### Email Handling
- **Priority Keywords**: urgent, asap, invoice, payment, deadline
- **Response Time**: Flag emails with priority keywords within 1 hour
- **Tone**: Professional and concise
- **Never Auto-Send**: Always create drafts for human review

### General Principles
- Always be polite and professional
- When in doubt, ask for human approval
- Log all actions taken
- Never delete original messages

## Task Processing Rules

### Priority Levels
1. **High**: Contains keywords: urgent, asap, deadline, payment
2. **Medium**: Client communications, invoices, scheduling
3. **Low**: Newsletters, updates, general inquiries

### Action Thresholds
- **Auto-Process**: Reading and categorizing emails
- **Requires Approval**: Sending emails, making payments, scheduling meetings

## Error Handling
- If uncertain about intent, create a task in /Needs_Action with [REVIEW NEEDED] tag
- Log all errors to /Logs folder
- Never retry failed actions more than 3 times
